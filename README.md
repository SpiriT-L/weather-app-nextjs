# 🌦️ Weather App with Next.js

A modern web application that provides users with up-to-date weather information for any selected city. Built with the **Next.js** framework, it delivers high performance, server-side rendering, and excellent SEO optimization.

## 🔧 Key Features

- City search and current weather display  
- Temperature, humidity, pressure, wind speed  
- Multi-day weather forecast  
- Geolocation support for automatic location detection  
- Responsive design for mobile and desktop devices

## 🛠️ Technologies Used

- **Next.js** — SSR and routing  
- **React** — UI components  
- **TypeScript** — static typing  
- **Tailwind CSS** — styling  
- **OpenWeatherMap API** — weather data source

# 🌦️ Weather App на Next.js

Современное веб-приложение, предоставляющее пользователю актуальную информацию о погоде в выбранном городе. Построено на фреймворке **Next.js**, что обеспечивает высокую производительность, серверный рендеринг и отличную SEO-оптимизацию.

## 🔧 Основные функции

- Поиск города и отображение текущей погоды
- Температура, влажность, давление, скорость ветра
- Прогноз на несколько дней
- Геолокация для автоматического определения местоположения
- Адаптивный дизайн для мобильных и десктопных устройств

## 🛠️ Используемые технологии

- **Next.js** — SSR и маршрутизация
- **React** — UI-компоненты
- **TypeScript** — типизация
- **Tailwind CSS** — стилизация
- **OpenWeatherMap API** — источник погодных данных
